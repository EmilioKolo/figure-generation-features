# FigGen DNAseq

 Utilities to generate paper-ready figures from different data. Designed with a DNA sequencing and machine learning pipeline in mind.

# Requirements

The pipeline has been designed with the following versions of particular programs and libraries:
* Python3 == 3.13.9
* pip == 25.3
    * matplotlib == 3.10.7
    * numpy == 2.3.5
    * pandas == 2.3.3
    * seaborn == 0.13.2
